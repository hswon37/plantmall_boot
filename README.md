# plantmall_boot
plantmall for Spring Boot

=======
1. 주제
- 식물 판매 웹사이트

2. 핵심 요약<br>
  (1) 문제 배경<br>
  \- 코로나-19로 집에서 보내는 시간이 늘면서 식물 키우기 취미 열풍, 식물 관련 매출 증가<br>
  (2) 해결 방안<br>
  \- 편리하게 식물을 조회하고 판매할 수 있는 식물 판매 웹사이트 제작<br>
  (3) 결론<br>
  \- 식몰(식물+Mall) 사이트 개발 <br><br>

3. 기여한 점
- 4인 팀프로젝트
- 제품 판매, 구매, 서비스 개발 역할
- 업무: 판매 제품 관리, 제품 조회, 검색, 홈, 메뉴바

4. 사용 기술
- Spring Boot
- Oracle
- Erwin
- Mybatis
